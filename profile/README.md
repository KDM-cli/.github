# KDM

KDM develops modern, high-performance command-line tools designed for seamless Docker and Kubernetes monitoring. 

Focused on speed, simplicity, and developer experience, KDM provides lightweight, terminal-first solutions for monitoring containers, clusters, and infrastructure health in real time.

---

## 📢 Announcement: Join the Core Team!

We are actively looking for passionate maintainers to join the organization and help shape the future of our CLI tools. 

### How to Join:
* **Contribute:** Submit **5 or more meaningful Pull Requests (PRs)**.
* **Focus Areas:** We are looking for improvements to the core CLI functionality, optimizations, or enhancements to our GitHub Actions workflows.
* **The Reward:** Contributors who meet these criteria and show a strong understanding of the project will receive an official invitation to join the **KDM-cli** organization as a core maintainer.

We welcome your ideas and look forward to building better developer tools together!
